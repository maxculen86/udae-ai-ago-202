# Trabajo Practico Obligatorio Aplicaciones interactivas

## Consultorio Medico

### Prerequisitos

Node
npm
Express
jsonwebtoken
mongoose

### Clone

- Clone this repo to your local machine using [https://github.com/Abdurraheem/REST-API-JWTWEB-TOKEN.git]

### Setup

> Instalar backend

```shell
$ cd api-ago-2020
$ cd backend
$ npm install
```
> Instalar frontend

```shell
$ cd api-ago-2020
$ cd frontend
$ npm install
```
## Correr backend y frontend
> Ejecutar backend

```shell
$ cd api-ago-2020
$ cd backend
$ npm install
```
> Ejecutar frontend

```shell
$ cd api-ago-2020
$ cd frontend
$ npm install
```
